How to Use DataVista

1

Upload Your Data

Select a CSV file containing your data. The file should have headers in the first row and can contain any number of rows and columns.

2

Configure Settings

Customize your report title, subtitle, and choose which visualizations to include in your analysis.

3

Generate Report

Click the "Generate Report" button to start the analysis. The process runs in the background and you can monitor progress.

4

Review Results

Once complete, open the generated PDF report to view comprehensive analysis, visualizations, and recommendations.
